# Infiraft Site

Official landing page for Infiraft — a small website and digital product studio.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The static output will be generated in `dist/`.

## Deploy notes

### Cloudflare Pages

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`

### Vercel

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`
